# projet-vlan-routage-wifi-radius
This project designs and configures a full network infrastructure with VLANs, inter-VLAN routing, DHCP, RADIUS, and secure Wi-Fi. Three VLANs are used: VLAN 10 (sale/key_sale), VLAN 20 (pa-direction with RADIUS auth), and VLAN 200 (management). R1 routes and serves DHCP, S1 trunks, AP bridges.
